# Yoania

This component's mission is to control the buffering state of all playable content that requires synchronization.

## Dependencies

- [klass](https://github.com/ded/klass)

## Dev Dependencies

- gulp
- gulp-jshint

## Running Tests

Open the file tests/yoania.html in your browser.

## Check code quality

`gulp lint`

## Minify

`gulp minify`
